# JYS마케팅 브랜드 가이드라인

> 이 문서는 JYS마케팅의 브랜드 정체성을 일관되게 유지하기 위한 공식 가이드입니다.  
> 모든 내·외부 커뮤니케이션에서 반드시 준수해야 합니다.

---

## 1. 브랜드 아이덴티티

### 1-1. 브랜드 철학

JYS마케팅은 **"작은 브랜드도 빛날 수 있다"** 는 믿음을 핵심 철학으로 삼습니다.

우리는 예산의 크고 작음이 마케팅 성과를 결정하지 않는다고 믿습니다.  
**전략의 정밀함**, **실행의 일관성**, **사람에 대한 이해**가 성과를 만든다고 생각합니다.

### 1-2. 브랜드 미션

> 대한민국 중소기업과 소상공인이 마케팅으로 인해 포기하거나 손해 보는 일이 없도록 한다.

### 1-3. 브랜드 비전

> 2030년까지 국내 소상공인·중소기업 마케팅 대행 분야에서  
> **신뢰도 1위 파트너**로 자리매김한다.

### 1-4. 핵심 가치 (Core Values)

| 가치 | 설명 |
|------|------|
| **신뢰 (Trust)** | 계약서에 없는 것도 먼저 챙기는 파트너십 |
| **실용 (Practicality)** | 이론이 아닌 매출로 증명하는 마케팅 |
| **공감 (Empathy)** | 고객의 사업을 내 사업처럼 이해하는 태도 |
| **성장 (Growth)** | 고객이 성장해야 우리도 성장한다는 철학 |

## 2. 브랜드 퍼스낼리티

### 2-1. 브랜드 성격 키워드

```
믿음직한  ·  친근한  ·  전문적인  ·  실용적인  ·  진솔한
```

### 2-2. 브랜드 톤 & 보이스 (Tone & Voice)

JYS마케팅의 커뮤니케이션은 항상 아래 원칙을 따릅니다.

**✅ 이렇게 말합니다**
- 쉽고 명확하게: 마케팅 전문 용어를 풀어서 설명한다
- 따뜻하게: 고객의 어려움에 공감하는 언어를 사용한다
- 솔직하게: 과장 없이 현실적인 기대치를 제시한다
- 능동적으로: "해드릴게요"보다 "함께 만들겠습니다"

**❌ 이렇게 말하지 않습니다**
- 지나친 업계 용어 남발 (ROAS, CTR 등을 설명 없이 사용)
- 과도한 보장성 표현 ("무조건 1등", "반드시 매출 상승")
- 거만하거나 가르치는 듯한 어조
- 고객을 어리숙하게 여기는 표현

### 2-3. 페르소나 이미지

> JYS마케팅을 한 사람에 빗대면:  
> **"경험 많은 동네 선배 창업자"**  
> — 화려하진 않지만 실전에서 단련된 조언을 건네며, 내 사업을 진심으로 응원해주는 사람.

## 3. 로고 가이드라인

### 3-1. 기본 로고 구성

```
[심볼] + [워드마크]

심볼: 상승하는 화살표와 J·Y·S 이니셜을 결합한 마크
워드마크: "JYS마케팅" (국문) / "JYS Marketing" (영문)
```

### 3-2. 로고 사용 원칙

| 항목 | 규칙 |
|------|------|
| **최소 사용 크기** | 디지털: 80px 이상 / 인쇄: 20mm 이상 |
| **여백 (Clear Space)** | 로고 높이의 1/4 이상 사방 확보 |
| **배경색** | 화이트 또는 브랜드 네이비 위에만 사용 |
| **변형 금지** | 색상 임의 변경, 비율 왜곡, 회전, 그림자 추가 금지 |

### 3-3. 로고 변형 버전

| 버전 | 사용 상황 |
|------|----------|
| **풀컬러 (기본형)** | 밝은 배경의 일반적인 용도 |
| **화이트 역전형** | 어두운 배경, 사진 위 배치 시 |
| **모노크롬 (흑백)** | 팩스, 흑백 인쇄, 각인 등 |
| **심볼 단독형** | SNS 프로필 이미지, 파비콘, 아이콘 등 소형 활용 |

## 5. 타이포그래피

### 5-1. 국문 폰트

| 구분 | 폰트명 | 용도 |
|------|--------|------|
| **메인** | Noto Sans KR | 본문, UI 전반 |
| **헤드라인** | Noto Sans KR Bold | 제목, 강조 문구 |
| **서브** | Spoqa Han Sans Neo | 캡션, 태그, 보조 텍스트 |

### 5-2. 영문 폰트

| 구분 | 폰트명 | 용도 |
|------|--------|------|
| **메인** | Inter | 영문 본문, UI |
| **헤드라인** | Inter Bold / ExtraBold | 영문 제목 |
| **모노스페이스** | JetBrains Mono | 수치, 코드성 데이터 |

### 5-3. 폰트 사이즈 스케일

```
Display  : 40px / Bold   — 메인 히어로 문구
H1       : 32px / Bold   — 페이지 제목
H2       : 24px / SemiBold — 섹션 제목
H3       : 20px / Medium — 소제목
Body     : 16px / Regular — 본문
Small    : 14px / Regular — 보조 설명
Caption  : 12px / Regular — 캡션, 주석
```

### 5-4. 타이포 사용 원칙

- 한 페이지 내 폰트 굵기는 **최대 3단계** 이내로 사용
- 밑줄은 **하이퍼링크 전용**으로만 사용
- 글자 색상은 브랜드 컬러 시스템 내에서만 선택
- 줄 간격(Line Height): 본문 기준 **1.6~1.8배**

## 6. 이미지 & 비주얼 스타일

### 6-1. 이미지 방향성

**사용하는 이미지**
- 실제 현장감이 느껴지는 사진 (소상공인의 일상, 매장, 제품)
- 사람이 등장하는 따뜻한 분위기의 사진
- 밝고 자연광 위주의 구도
- 한국적 맥락이 담긴 생활 이미지

**사용하지 않는 이미지**
- 지나치게 연출된 스튜디오 느낌의 스톡 이미지
- 외국인 중심 이미지 (타겟 고객과 괴리)
- 어둡거나 무거운 톤의 사진
- 클리셰한 악수 / 화이트보드 / 정장 회의 이미지

### 6-2. 그래픽 요소

- **일러스트**: 플랫 디자인 스타일, 브랜드 컬러 기반
- **아이콘**: 라인 아이콘 계열, 2px 스트로크 기준
- **차트/데이터 시각화**: 브랜드 컬러 팔레트 내에서 구성
- **패턴/텍스처**: 과도한 장식 지양, 단순한 기하 패턴만 허용

### 6-3. 콘텐츠 이미지 비율

| 플랫폼 | 권장 비율 |
|--------|----------|
| 인스타그램 피드 | 1:1 또는 4:5 |
| 인스타그램 스토리 / 릴스 | 9:16 |
| 블로그 대표 이미지 | 16:9 |
| 카카오채널 포스트 | 1:1 또는 2:1 |
| 유튜브 썸네일 | 16:9 (1280×720px) |

## 7. SNS 채널별 커뮤니케이션 가이드

### 7-1. 인스타그램

- **톤**: 따뜻하고 공감 가는 언어, 이모지 적극 활용
- **해시태그**: 게시물당 10~15개 (브랜드 태그 + 업종 태그 + 지역 태그 조합)
- **고정 해시태그**: `#JYS마케팅` `#소상공인마케팅` `#마케팅대행`
- **게시 빈도**: 피드 주 3회 / 스토리 매일

### 7-2. 블로그 (네이버)

- **톤**: 정보성 · 전문성 · 친근함 균형
- **구조**: 문제 제시 → 원인 분석 → 해결책 → JYS 사례 → CTA
- **글자 수**: 최소 1,500자 이상
- **이미지**: 본문 내 최소 5장 이상 포함

### 7-3. 카카오 채널

- **톤**: 매우 짧고 명확하게, 핵심만 전달
- **CTA**: 매 메시지마다 상담 신청 링크 삽입
- **발송 빈도**: 주 1~2회 (과도한 발송 지양)

## 8. 명함 / 인쇄물 가이드

### 8-1. 명함

```
[앞면]
- 로고 (좌상단 또는 중앙)
- 이름 / 직책 (Bold, 18pt 이상)
- 연락처 · 이메일 · 웹사이트

[뒷면]
- 브랜드 네이비 단색 배경
- 슬로건: "작은 브랜드도 빛날 수 있습니다"
- QR코드 (웹사이트 또는 카카오채널 연결)

규격: 90mm × 50mm (국내 표준)
```

### 8-2. 제안서 / 보고서

- 표지: 브랜드 네이비 배경 + 흰색 타이포
- 내지: 흰색 배경, 네이비·오렌지 포인트 컬러
- 폰트: Noto Sans KR 일관 사용
- 여백: 상하좌우 최소 20mm 이상 확보

## 9. 금지 사항 (Brand Don'ts)

| 금지 항목 | 이유 |
|-----------|------|
| 로고 색상 임의 변경 | 브랜드 일관성 훼손 |
| 브랜드 컬러 외 형광색 사용 | 브랜드 톤 이탈 |
| 경쟁사 비방성 문구 | 브랜드 신뢰도 손상 |
| 보장성 과장 광고 문구 | 법적 리스크 및 신뢰 손상 |
| 저화질 이미지 사용 (72dpi 미만 인쇄) | 브랜드 품질 저하 |
| 무단 폰트 사용 | 저작권 문제 |
| 로고 위에 텍스트 또는 이미지 중첩 | 가독성 및 식별성 저하 |
